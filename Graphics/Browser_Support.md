# GraphicsJS Browser Support

GraphicsJS comes packaged with SVG and VML renderers. GraphicsJS rely on Google Closure Library in full compliance with Google JavaScript Style Guide. 

## Browser Support

* Internet Explorer 6.0+
* Firefox 2.0+
* Chrome 1.0+
* Safari 4.0+
* Opera 9.0+
* iOS 3.0+
* Android 3.0+
